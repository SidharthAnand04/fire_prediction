For sake of demonstration, a person can manaully enter data. If implemented it would be done automatically via an API.

Information Guide

X, Y: Coordinates within the Montesinho park map (1-9).

Month, Day: Month and day of the week the observation was made.

FFMC: Fine Fuel Moisture Code index from the FWI system.

DMC: Duff Moisture Code index from the FWI system.

DC: Drought Code index from the FWI system.

ISI: Initial Spread Index from the FWI system.

Temp: Temperature in Celsius degrees.

RH: Relative humidity in %.

Wind: Wind speed in km/h.

Rain: Outside rain in mm/m2.
